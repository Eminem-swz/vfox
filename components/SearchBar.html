# SearchBar 搜索栏

  • 该组件本身没有固定顶部功能，可以配合 Sticky 组件实现置顶功能。

# Props

属性类型默认值必填说明
placeholdersstring/string[]否设置候选值列表，多个时轮询显示
placeholder-intervalnumber5000否placeholder 切换时间，placeholders 有多个时生效
ghostbooleanfalse否是否开启反色模式，开始后适用于深色底色，通过 CSS 设置背景色
readonlybooleanfalse否是否只读模式，开启后主要用于搜索入口场景
maxlengthnumber100否最大长度
focusbooleanfalse否加载时是否获取焦点
show-cancelbooleanfalse否是否展示取消按钮

# Events

事件描述回调函数参数
input输入值改变时触发，包括 clear 的({ text: string }, setSuggestList: suggestList => void)
focus输入框获取焦点时触发({ text: string }, setSuggestList: suggestList => void)
blur输入框移出焦点时触发({ text: string }, setSuggestList: suggestList => void)
cancel取消按钮点击({})
click输入框点击，配合 readonly 使用，获取到当前候选值({ searchText: string})

# setSuggestList 的参数 suggestList 结构

['suggest A', 'suggest B']

或者

[{
  text: '油烟机',
  tags: ['厨房', '电器']
}]
