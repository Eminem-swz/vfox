Progress 进度条 #

Import #

import { Progress } from 'vfox'

具体的引入方式可以参考引入组件。

Props #

属性类型默认值必填说明
percentagestring | number是百分比，例如：50
show-textbooleanfalse否是否展示文字
fixed-barbooleanfalse否是否固定进度条宽度，配合 show-text=true 使用，防止由于文字 5% 和 100% 宽度不一样导致进度条长短不一

Slots #

内容区（#default） #

<fx-progress :percentage="5">
  <template #default="{ progress }"> 已抢{{ progress }} </template>
</fx-progress>

注：添加 slot 后 show-text prop 属性失效。
