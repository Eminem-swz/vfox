Pagination 分页 #

Import #

import { Pagination } from 'vfox'

具体的引入方式可以参考引入组件。

Props #

属性类型默认值必填说明
v-model:currentnumber1否当前页码
totalnumber1否总页数

Events #

事件描述回调函数参数TypeScript 函数
change点击翻页按钮时触发payload: { current: number, total: number }PaginationOnChange

Slots #

默认（#default） #

<fx-pagination>
  <template #default="{ current, total }"> 第 {{ current }} / {{ total }} 页 </template>
</fx-pagination>

翻页按钮（#prev/#next） #

<fx-pagination>
  <template #prev> 上一页 </template>
  <template #next> 下一页 </template>
</fx-pagination>
