Scroll 滚动 #

Scroll.pageScrollTo(object) #

将页面滚动到目标位置。

Params #

Object object

属性类型默认值必填说明
scrollTopnumber否滚动到页面的目标位置，单位 px
durationnumber300否滚动动画的时长，单位 ms
successfunction否接口调用成功的回调函数
failfunction否接口调用失败的回调函数
completefunction否接口调用结束的回调函数（调用成功、失败都会执行）

Usage #

import { Scroll } from 'vfox'

Scroll.pageScrollTo({
  scrollTop: 0,
  duration: 300
})

Scroll.scrollTo(object) #

Params #

Object object

属性类型默认值必填说明
selectorelement/string是要滑动的元素，如果是 string，则为可以被 document.querySelector(selector) 获取到
scrollTopnumber否滚动到页面的目标位置，单位 px
durationnumber300否滚动动画的时长，单位 ms
successfunction否接口调用成功的回调函数
failfunction否接口调用失败的回调函数
completefunction否接口调用结束的回调函数（调用成功、失败都会执行）

Usage #

Scroll.scrollTo({
  selector: document.documentElement
  scrollTop: 0,
  duration: 300
})
