指南 #

介绍 #

移动端 Vue 3.x 组件库。

特性 #

  • 提供 70+ 套组件，覆盖移动端各类场景
  • 单元测试覆盖
  • 支持 Vue 2 和 Vue 3
  • 支持按需引入
  • 支持主题定制
  • 支持黑暗模式
  • 支持国际化
  • 支持 TypeScript

附言 #

该版本已使用 Composition API 和 Typescript 重构。

安装 #

# npm
npm i -S vfox

# yarn
yarn add vfox

# pnpm
pnpm add vfox
